About ReliaSuite Professional

A professional reliability engineering platform built for the field.

ReliaSuite Professional is a browser-based reliability analysis application designed specifically for oil & gas and mining operations. It brings rigorous statistical reliability methods to maintenance engineers and reliability professionals who need fast, accurate, auditable results.

Key Features

  • Life Data Analysis — Weibull, Lognormal, Exponential, Mixed Weibull
  • Reliability Growth — Crow-AMSAA NHPP, Crow Extended
  • Degradation Analysis — Linear, Power, Wiener, Gamma Process models
  • Preventive Maintenance & FFI Optimization (Barlow-Hunter)
  • Spare Parts Forecasting (Poisson / Binomial / Normal)
  • Report generation for every analysis type
  • Multi-Asset / Fleet Analysis with tagging
  • Portfolio Dashboard with KPI rollup

Target Audience

Designed for reliability engineers, maintenance planners, and asset managers in heavy industry sectors including oil & gas, mining, and power generation.

Glossary

R(t) — Reliability Function

The probability that an item performs its required function without failure under stated conditions for a specified time interval t. Also called the survival function.

F(t) — Cumulative Distribution Function (Failure CDF)

The probability that a failure occurs by time t. F(t) = 1 − R(t).

h(t) — Hazard Rate / Failure Rate

The instantaneous rate of failure at time t given survival to t. Also called the hazard function or force of mortality.

MTBF — Mean Time Between Failures

The average time between successive failures of a repairable system. Equal to the integral of R(t) for a non-repairable item.

MTTR — Mean Time To Repair

The average time required to repair a failed component and return it to operational status.

Weibull Distribution

A flexible two-parameter (or three-parameter) lifetime distribution. The shape parameter β distinguishes infant mortality (β<1), random failures (β≈1), and wear-out (β>1).

Crow-AMSAA (NHPP Power Law)

A Non-Homogeneous Poisson Process model used for reliability growth analysis of repairable systems under test or in service.

Degradation Analysis

A method to estimate time-to-failure from degradation measurements (e.g. vibration, corrosion, wear) when direct failure data is scarce. Includes linear, power, Wiener stochastic, and Gamma process models.

RUL — Remaining Useful Life

The expected time remaining before a degrading item reaches a predefined failure threshold.

FFI — Failure Finding Interval

The optimal inspection interval for hidden failures (protective devices, stand-by equipment) calculated to achieve a target unavailability.

Confidence Bounds

Statistical intervals reflecting uncertainty in parameter estimates. ReliaSuite supports Fisher Matrix and Likelihood Ratio bounds.

β (Beta) — Shape Parameter

Weibull shape parameter controlling the failure rate behavior over time. β<1: decreasing rate (infant mortality); β=1: constant rate; β>1: increasing rate (wear-out).

η (Eta) — Scale Parameter / Characteristic Life

Weibull scale parameter representing the age at which 63.2% of the population has failed.

Methodology & References

Statistical Methods

ReliaSuite implements Maximum Likelihood Estimation (MLE) and Rank Regression (RRX/RRY) following methods documented in ReliaSoft's Life Data Analysis Reference and the textbooks referenced below.

Probability Plotting

Johnson adjusted ranks (Bernard's approximation with suspension adjustment per ReliaSoft Eq. 8.2) are used to correctly account for censored data in all probability plots.

Confidence Bounds

Fisher Matrix bounds follow ReliaSoft Equations 519, 533, 536, 538, 563. Likelihood Ratio bounds use profile likelihood contours. Crow confidence bounds follow Crow (1990) as documented in ReliaSoft's RGA Reference.
